数组它是由一个固定长度的特定类型元素组成的序列。

因为数组的长度是固定的，所以在go语言中很少使用数组，常常使用切片。

数组的声明方式：

```go
// 第一种
var arr [2]int
arr[0] = 1
arr[1] = 2
// 第二种
var arr = [2]int{1, 2}
// or
arr := [2]int{1, 2}
// 第三种
var arr = [...]int{1, 2}
//or
arr := [...]int{1, 2}
// 第四种
var arr = [...]int{1, 2}
//or
arr := [...]int{1, 2}
```

在go语言中没有被初始化的元素，会被默认初始化成对应类型的零值
